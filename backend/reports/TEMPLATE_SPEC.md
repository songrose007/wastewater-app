## 六、流程图格式

### 6.1 流程图标题
```
图3-1  废水处理工艺流程图                       ← 黑体 10pt，居中，加粗
```

### 6.2 流程图本体
- **推荐格式**：三列并行的圆角矩形箭头图（ProcessOn 风格）
  - 左列：生化处理（蓝色系）
  - 中列：主流工艺（深蓝色系）
  - 右列：浓水/深度处理（橙色系）
- 框体：灰色填充（#F0F0F0）、深灰边框（#666666）、圆角 6px
- 连接线：1.5px 实线，带三角箭头
- 回流线：虚线（#999999）
- 药剂标注：在对应单元右侧，灰色小字
- **嵌入方式**：生成 PNG（1400x780px），嵌入 DOCX 居中显示
- **备用方案**：若无法生成图片，使用合并单元格表格模拟流程图

## 七、构筑物及设备描述格式

每个构筑物按以下五要素格式描述：

```
（1）原水调节池                                   ← 构筑物名称
数量：1座                                          ← 数量（冒号后 Tab 对齐）
结构形式：地下式钢砼防腐                            ← 结构形式
池体尺寸：8.0 m × 4.0 m × 4.0 m（有效水深3.5 m，有效容积约100 m³）  ← 尺寸
设计参数：停留时间 8 h                              ← 设计参数
配套设备1：提升水泵 2 台。Q=12 m³/h，H=15 m，N=1.1 kW，自吸泵，过流 304 不锈钢。
配套设备2：穿孔曝气搅拌系统 1 套。DN80，ABS 材质。
配套设备3：液位控制系统 1 套（浮球液位计，0~4 m）。
```

**格式要点**：
- 构筑物名称前用 `（序号）` 编号
- 每个设备描述一行：`名称 + 数量 + 台/套。参数：`
- 设备参数用逗号分隔，关键参数标注单位
- 材质标注在末尾（如 `过流 304 不锈钢`）

## 八、工艺描述格式

工艺流程描述使用编号分段：

```
（1）废水经收集管道进入原水调节池（有效容积约100 m³，HRT=8 h），
    池内设穿孔曝气搅拌装置，对来水进行均质均量调节。
    调节池出水通过提升泵送至综合调节池。

（2）综合调节池（有效容积约150 m³，HRT=12.5 h）进一步均化水质，
    同时接纳 MVR 蒸发冷凝水等回流液。池内设穿孔曝气搅拌系统，
    必要时投加 NaOH 调节 pH。
```

**格式要点**：
- 使用 `（1）（2）（3）...` 全角括号编号
- 每一步描述一段完整的工艺过程
- 关键设计参数在括号中标注
- 化学反应方程式用标准化学式表示
