# WebUI 方案生成工作流

本地启动后，打开：

```text
http://localhost:5173
```

## 推荐使用流程

1. 新建项目。
2. 进入“方案向导”。
3. 按“待处理事项”补充项目基础信息和水质参数。
4. 运行工艺推荐，并在工艺选择页确认路线。
5. 在设计计算页审查或修改经验参数，然后执行计算。
6. 上传平面图/高程图 PDF。
7. 在构筑物映射页确认图纸元素与构筑物参数的映射。
8. 在设计校核页查看图纸参数和计算参数的对比。
9. 执行设备选型和造价估算。
10. 生成 HTML/PDF/DOCX 报告。
11. 生成成果包 ZIP。

## 交互式确认原则

WebUI 会集中显示：

- 必须填写的输入；
- 必须确认的工艺路线；
- 设计计算警告；
- 图纸映射待处理项；
- 设备选型与造价估算动作；
- 报告生成与成果打包动作。

复杂修改仍在原有专业页面完成，方案向导负责总控、提示和跳转。

## 输出文件

报告页和方案向导可以下载：

- PDF 方案；
- DOCX 方案；
- HTML 预览；
- 项目成果 ZIP 包。
